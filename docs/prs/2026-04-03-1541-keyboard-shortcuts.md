## Shortcuts

| Key | Action |
|-----|--------|
| `/` | Focus search |
| `Ctrl+K` / `Cmd+K` | Clear all requests |
| `Ctrl+E` / `Cmd+E` | Toggle capture |
| `Ctrl+Shift+C` | Copy selected as cURL |
| `Enter` | Open selected request detail |
| `Esc` | Close detail panel |
| `Tab` | Switch Request/Response tab |
| `B` | Toggle breakpoint on selected |
| `R` | Open resend for selected |
| `?` | Show keyboard shortcuts help |

## Notes

- Single-key shortcuts (/, ?, B, R, Tab, Enter) are suppressed when focus is in an input/textarea/select
- Ctrl/Cmd shortcuts work regardless of focus context

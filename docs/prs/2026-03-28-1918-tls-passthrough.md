## Description

Add TLS passthrough for configured domains. When shmoxy MITM-intercepts HTTPS traffic, it
changes the TLS fingerprint (JA3/JA4), causing some services like Cloudflare to reject requests.
This feature allows configuring domains to be tunneled as raw TCP without TLS termination,
preserving the client's original TLS fingerprint.

Closes #99
